# Avionti Repository

This repository contains projects and websites for Avionti.

## Projects

### PicToPDF Website
- **Location**: `/PicToPDF/`
- **Description**: Marketing website for the PicToPDF mobile app
- **Live URL**: [avionti.vercel.app](https://avionti.vercel.app)
- **Technology**: Static HTML/CSS/JS website

## Deployment

The repository is configured to deploy the PicToPDF website to Vercel automatically:

- **Domain**: `avionti.vercel.app`
- **Source**: `/PicToPDF/` folder
- **Auto-deploy**: Enabled on push to `main` branch

## Development

To work on the PicToPDF website:

1. Navigate to the `PicToPDF` folder
2. Make your changes to HTML/CSS/JS files
3. Test locally by opening `index.html` in a browser
4. Commit and push to `main` branch for automatic deployment
